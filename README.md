# images

Markdown 文档编写时可使用本地图片，但是无法在网络上使用。图床的作用可以理解为将文档中的图片放到网络上，直接引用网络地址，这样可以做到无论在那个平台都可以使用统一的图片地址。

自定义域名：使用了 CDN 加速，配置格式为：

```xml
https://cdn.jsdelivr.net/gh/用户名/仓库名
```
